# Inverurie Probabilistic Flood Explorer

Probabilistic flood mapping for the River Don at Inverurie, Scotland,
using a surrogate model with Monte Carlo uncertainty quantification.

**UQ4FM project — University of Edinburgh**

Nina Fischer, Adil Siripatana, Amy Wilson, Lindsay Beevers

## About

- 1-in-100 year return period flood scenario
- LISFLOOD-FP ensemble simulations
- POD-PCK surrogate model
- Monte Carlo uncertainty propagation (N=500)
- Study area: Inverurie, Aberdeenshire, Scotland
